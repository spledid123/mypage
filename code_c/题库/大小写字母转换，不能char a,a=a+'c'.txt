/*1:大小写字母互换
描述
把一个字符串中所有出现的大写字母都替换成小写字母，同时把小写字母替换成大写字母。

输入
输入一行：待互换的字符串。
输出
输出一行：完成互换的字符串（字符串长度小于80）。
样例输入
If so, you already have a Google Account. You can sign in on the right. 
样例输出
iF SO, YOU ALREADY HAVE A gOOGLE aCCOUNT. yOU CAN SIGN IN ON THE RIGHT. 
*/
//转换大小写 ,32，不能用('A'-'a') 
#include<stdio.h>
#include<string.h>

int main(){
	char a[100];
	gets(a);
	int i,n;
	n=strlen(a);
	for(i=0;i<n;i++){
		if('A'<=a[i]&&a[i]<='Z'){
		a[i]=a[i]+32;}
		else if('a'<=a[i]&&a[i]<='z'){
		a[i]=a[i]-32;}
		
	}

	printf("%s",a);
}
